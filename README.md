# 키친포스

## 퀵 스타트

```sh
cd docker
docker compose -p kitchenpos up -d
```

## 요구 사항
- 메뉴 그룹
    - [ ]  새로운 메뉴 그룹 등록.
    - [ ]  메뉴 그룹 전부 가져오기.
- 메뉴
    - [ ]  메뉴의 등록.
    - [ ]  메뉴의 가격을 수정한다.
    - [ ]  메뉴 전시 상태 ON
    - [ ]  메뉴 전시 상태 OFF
    - [ ]  메뉴 전부 가져오기
- 주문 테이블
    - [ ]  주문 테이블 등록.
    - [ ]  테이블 앉히기
    - [ ]  테이블 치우기
    - [ ]  테이블 고객수 변경하기
    - [ ]  주문 테이블 전부 가져오기.
- 주문
    - [ ]  주문 등록.
    - [ ]  주문 받기
    - [ ]  주문 서빙
    - [ ]  배달 시작하기
    - [ ]  배달 완료
    - [ ]  주문 완료
    - [ ]  주문 전부 가져오기.
- 상품
    - [ ]  상품 등록.
    - [ ]  상품 가격 수정하기.
    - [ ]  상품 전부 가져오기.
## 용어 사전

| 테이블       | 영문명 | 설명                                                             |
|-----------| --- |----------------------------------------------------------------|
| 메뉴        | menu  | 메뉴 정보(노출 여부, 이름, 가격), 메뉴그룹(menu_group), 메뉴상품(menu_product) 리스트 |
| 메뉴 그룹     | menu_group  | 메뉴를 그룹화하여 명칭 부여 ex) 추천메뉴                                       |
| 메뉴 상품     | menu_product  | 수량, 상품ID, 메뉴ID                                                 |
| 주문 라인 아이템 | order_line_item  | 수량, 메뉴ID, 주문ID                                                 |
| 주문 테이블    | order_table  | 주문 테이블. 이름, 고객 수, 가득참 여부                                       |
| 주문        | orders  | 주문 정보(배달 주소, 주문 시각, 상태, 타입)                                    |
| 상품        | product  | 상품 정보(이름, 단일 가격)                                               |


| Enum 정의 | 영문명 | 설명 |
|---------| --- | --- |
| 주문 상태   | OrderStatus  | WAITING, ACCEPTED, SERVED, DELIVERING, DELIVERED, COMPLETED  |
| 주문 타입   | OrderType  | DELIVERY, TAKEOUT, EAT_IN  |
